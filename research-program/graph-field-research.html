Graph-field research

This preview summarizes the current research program behind aimez: measured fields on graphs, local edge values, route cost, loop behavior, and the conditions under which a local measurement becomes larger structure.

Core object

The current project uses a Manhattan pedestrian graph as the concrete test domain for these ideas. Camera-derived crowd pressure and capacity layers assign route-cost values to directed edges and adjacent corridor segments.

Structural test

Route comparisons show how local edge assignments change the cost and exposure of a chosen path. Loop closure then tests whether those changes cancel around the graph or leave residual structure that emerges only at the graph level.

Program role

The Manhattan case is one applied instance of a broader research interest in measured systems, spatial reasoning, simulation, decision surfaces, human-centered AI, and research tooling.
