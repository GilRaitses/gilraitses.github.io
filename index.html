Research focus

I build computational infrastructure for multi-scale analysis across behavioral and sensor-driven systems, with an emphasis on representational coherence under variation in scale, resolution and structure.

The work blends neuroethology, signal processing and simulation with a pragmatic goal of keeping interpretability as internal descriptions shift.

Framework implementations

aimez.ai

Applied research program that takes public traffic-camera frames through computer-vision models into a camera-derived crowd pressure field and into pedestrian routing that respects lived burden. It carries the work from raw field data to a public site with notes, figures and validation briefs.

Central Casting

Checkpoint-based orchestration and audit for long, multi-repository agent work. Agent homes act as role contracts, checkpoints are the unit of memory and an alignment check flags when the structured record and the human-readable views diverge.

Current work

  • aimez.ai (Klosr) builds a city-scale evidence model for pedestrian conditions on the Manhattan walk graph, with comfort-aware routing as one application over fused camera and environmental layers.
  • magniphyq is a modular research visual workspace, a node graph that wires data, analysis and simulation into one runnable, inspectable pipeline, deployed as a live demo.
2024 to 2026
  • Drosophila sensory circuit analysis provides pipelines and visualization tools for sensory integration across gustatory, olfactory, mechanosensory and visual modalities, with temporal pattern extraction from trajectory data.
  • Marine behavioral states uses lightweight recurrent modeling for humpback whale behavioral states from dive telemetry, targeting multi-scale segmentation and event detection.
  • INDYsim is a hierarchical simulation framework for behavior as the interface between perception and adaptation, using agent based and event driven modeling of transitions under constraints.

aimez

aimez builds evidence models and tools for decisions about complex physical systems. The current product focus is Klosr, a city-scale evidence model of Manhattan that turns sparse observations into decision-ready pedestrian representations, with comfort-aware routing as one application on that surface.

aimez.ai  ·  Klosr  ·  whitepaper (PDF)  ·  repo

magniphyq

magniphyq is a modular research visual workspace where a whole pipeline lives as a node graph a researcher can run, inspect and clone. It wires track extraction, behavioral segmentation and contour analysis into one runnable graph, and it hosts a synapsin condensate phase-diagram campaign, so a measured model runs and compares against data in the same place. It is built and verified, deployed as a live demo.

Gil Raitses · 2026

Five comfort routing cost encodings on the Manhattan island walk graph. Comfort and adaptive routes use per-bin MRT from a precomputed SOLWEIG model. Gap to live weather data is documented with closure paths. Product name for the live surface is Klosr.
